# Landing page screenshots

Drop these four image files here (exact filenames) and they'll appear on the
landing page automatically:

| File               | Shown as                          |
| ------------------ | --------------------------------- |
| `assets-list.png`  | Hero image (top of landing page)  |
| `asset-details.png`| "Every asset, fully documented"   |
| `register.png`     | "Get set up in minutes"           |
| `login.png`        | "Secure sign-in for every role"   |

Tips:
- PNG or JPG both work. Aim for ~1600px wide for a crisp hero.
- If a file is missing, a styled placeholder shows in its place — nothing breaks.
- To use different images, edit `HERO_SHOT` / `SHOWCASE` in `src/pages/Landing.jsx`.
